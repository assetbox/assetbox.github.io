SVG to React Component

Provides a way to create a common internal icon library.

The best way to use the icon-build feature is to create a common icon project or separate it into monorepos.

If you create a common icon project, you can use other projects and icons right away through npm or monorepo.

Installation​

The assetbox provides a way to code-share svg files over npm or monorepo.
This chapter converts svg files into React Component code.

  • pnpm
  • yarn
  • npm
$ pnpm install @assetbox/plugin-react-icon -D
$ yarn add @assetbox/plugin-react-icon -D
$ npm install @assetbox/plugin-react-icon --save-dev

Setup​

assetbox.config.cjs
const { react } = require("@assetbox/plugin-react-icon");

/**
 * @type {import('@assetbox/cli').AssetBoxScheme}
 */
const config = {
  ...
  iconBuild: {
    outDir: "dist", // Build Output Folder
    plugins: [react()],
  },
};

module.exports = config;

Usage​

  • pnpm
  • yarn
  • npm
$ pnpm assetbox icon-build
$ yarn assetbox icon-build
$ npm run assetbox icon-build

Build to support both esm and cjs.
